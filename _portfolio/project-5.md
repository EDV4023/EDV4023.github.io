Cyber-Quiz AI provides an infinite amount of cybersecurity related questions so that students are constantly learning. Adjustable difficulties foster a broader range of abilities. The website was built in python. The frontend was a multipage Streamlit website and the backend consisted of Google's Genai API for LLM capabilities, and a .txt file for a user database of scores.
